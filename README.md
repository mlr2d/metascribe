# Metascribe

*Current version:1.0.0*

Metascribe is a lightweight utility designed to extract information from unstructured files and store them in structured databases. It leverages Jinja2-style templating and Regex extraction to parse arbitrary text files, capture key-value pairs, and store them into a SQLite database.
